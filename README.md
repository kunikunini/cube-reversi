# NEO REVERSI (3D Cube Reversi)

A futuristic 3D Cube Reversi game built with Vanilla JS and Vite.

## Features
- **3D Gameplay**: Play Reversi on all 6 faces of a 3x3x3 cube.
- **PWA Support**: Installable on PC and Mobile as a standalone app.
- **Neon Design**: High-end cyber-punk aesthetics.
- **AI Opponent**: Easy, Normal, and Hard difficulty levels.
- **Sound Effects**: Adaptive BGM and UI sound effects.

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Vercel Deployment

1. **GitHub/GitLab/Bitbucket**: Push this repository to your preferred Git provider.
2. **Vercel**:
   - Create a new project in Vercel.
   - Select this repository.
   - Vercel will automatically detect the **Vite** project and configure the build settings (`npm run build` and `dist` output).
   - Click **Deploy**.

## PC Playability
This game is optimized for both mouse and touch.
- **Rotate Cube**: Swipe or drag the cube.
- **Install**: In Chrome/Edge, click the "Install" icon in the address bar to play as a PC app.
